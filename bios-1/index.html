Introduccion

La protección con contraseñas en la BIOS o UEFI nos ayudará a prevenir que haya usuarios no autorizados con acceso físico al sistema y haga cambios en la configuración de esta. Las principales razones por las que asignaremos contraseñas a la BIOS o UEFI:

  • Prevenir cambios a las configuraciones
  • Prevenir el arranque del sistema Para la BIOS Lenovo Legion 5 Pro 16ACH tendremos que entrar a la pestaña de seguridad tal y como se muestra en la imagen, desde esta pestaña realizaremos las distintas configuraciones de seguridad.

    CONTRASEÑA DE ADMINISTRADOR

    Para evitar la configuración de la BIOS/UEFI le asignaremos una contraseña de administrador, esta será requerida cada vez que queramos acceder a la configuración

    CONTRASEÑA DE ARRANQUE o USUARIO (Power On)

    Podemos hacer que requiera una contraseña para el arranque del equipo, para ello tendremos que haber establecido primero una contraseña de administrador para habilitar configurar la contraseña de usuario o arranque. (Debido a que a es una BIOS simulada no es posible realizar esta parte) Arranques Externos Es necesario proteger el equipo de arranques desde dispositivos externos ya que esto puede producir el arranque de sistemas operativos o herramientas que puedan cargar en memoria. Para ello tendremos que acceder a la pestaña de boot Desde esta pestaña podremos deshabilitar el arraque desde un dispositivo externo, de forma predeterminada suele venir esta opción como habilitada por lo que la deshabilitaremos para aumentar la seguridad de nuestro equipo. Podremos desactivar los distintos arranques:
  • Arranque por USB (USB boot)
  • Arranque PXE En el caso que tengamos activado el arranque PXE podemmos activar que sea mediante IPV4 o IPV6. Orden de arranque Podemos seleccionar un orden de arranque para que tenga prioridad el dispositivo que nos interese paa que arranque por este de forma automatica. (Debido a que a es una BIOS simulada no es posible realizar esta parte) Otras opciones de seguridad

    Secure Boot

    Habilitar la opción de Secure Boot impedira el arranque de aplicaciones a través de dispositivos USB.

    Configuracion de puertos I/O

    Desde la pestaña de configuracion de los puertos I/O podemos encontrar distintos puertos. Entre ellos podemos encontrar algunos que pueden ser interesante como:
    • Ethernet LAN
    • Wireless WAN
    • Bluetooh
    • Puerto USB
    • Camara integrada
    • NFC
