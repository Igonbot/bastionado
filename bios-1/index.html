Introduccion

La protección con contraseñas en la BIOS o UEFI nos ayudará a prevenir que haya usuarios no autorizados con acceso físico al sistema y haga cambios en la configuración de esta. Las principales razones por las que asignaremos contraseñas a la BIOS o UEFI:

  • Prevenir cambios a las configuraciones
  • Prevenir el arranque del sistema Para la BIOS Insyde F.37 tendremos que entrar a la pestaña de seguridad tal y como se muestra en la imagen, desde esta pestaña realizaremos las distintas configuraciones de seguridad.

    CONTRASEÑA DE ADMINISTRADOR

    Para evitar la configuración de la BIOS/UEFI le asignaremos una contraseña de administrador, esta será requerida cada vez que queramos acceder a la configuración

    CONTRASEÑA DE ARRANQUE o USUARIO (Power On)

    Para evitar el aarranque le asignaremos una contraseña, esta será requerida cada vez que queramos arrancar el sistema Arranques Externos Es necesario proteger el equipo de arranques desde dispositivos externos ya que esto puede producir el arranque de sistemas operativos o herramientas que puedan cargar en memoria. Desde esta pestaña podremos deshabilitar el arraque desde un dispositivo externo, de forma predeterminada suele venir esta opción como habilitada por lo que la deshabilitaremos para aumentar la seguridad de nuestro equipo. Podremos desactivar los distintos arranques:
  • Arranque por USB (USB boot)
  • Arranque por la unidad de CD-ROM
  • Arranque de Red Si habilitamos el arranque por red podremos seleccionar si queremos utilizar IPV4 o IPV6 Orden de arranque Podemos seleccionar un orden de arranque para que tenga prioridad el dispositivo que nos interese para que arranque por este orden de prioridad de forma automatica. Otras opciones de seguridad

    Secure Boot

    Habilitar la opción de Secure Boot impedira el arranque de aplicaciones a través de dispositivos USB.
